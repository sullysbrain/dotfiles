# VSCode Plugins

Below are a list of plugins I use frequently in VSCode for productivity.

- Github Copilot (Microsoft)
- Github Chat (Microsoft)
- Vim (Vscodevim)
- Docker (Microsoft)
- C/C++ IntelliSense (Microsoft)
- Nord (an artic color theme, which I customized in settings.json)
- Pylance (Microsoft)
- Python - IntelliSense (Microsoft)
- Subtle Match Brackets (Rafa Mel)
- TabOut (Albert Romkes)
- PlatformIO (not used as often)
